# Tic‑Tac‑Toe

## Overview
A browser‑based **Tic‑Tac‑Toe** game with Player vs Player and Player vs CPU modes, 6 CPU difficulty levels, sounds, match summary, and saved progress via local storage.[2][3][5]

## How to Start
1. Put all files in one folder:  
   `index.html`, `style.css`, `script.js`, and all four `.mp3` sound files.[3][2]
2. **Start the project by opening (double‑clicking) `index.html` in your browser** (Chrome, Edge, Firefox, etc.).[3]

No extra setup, server, or installation is required.

## Gameplay
- Choose mode (Two Players or Vs CPU), target “First to (wins)”, player names, Player 1 mark, and CPU difficulty.[3]
- Click **Start Match** to show the board, scoreboard, and controls.[2][3]
- Click cells to play; CPU moves automatically in Vs CPU mode.[2]
- Use **Next Round**, **New Match**, **Reset Round**, and **Quit Match** to manage the match.[2][3]
